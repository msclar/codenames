Codenames
Play Codenames online across multiple devices on a shared board. To create a new game or join an existing game, enter a game identifier, language, and click the green button.

If you don't know how to play Codenames, this is a great resource.

All your gameplays will help the research community understand how humans communicate. So you'll be playing while helping science!
